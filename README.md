# 🧠 API de Análisis de Sentimientos (NLP)

Este proyecto es un microservicio basado en Inteligencia Artificial que analiza el sentimiento (positivo o negativo) de un texto en tiempo real. 

Está diseñado con una arquitectura cliente-servidor, donde esta API actúa como el motor cognitivo ("backend pesado"). Es ideal para integrarse en sistemas más grandes, como plataformas de atención al cliente o aplicaciones de administración, permitiendo clasificar y priorizar automáticamente el *feedback* de los usuarios.

## 🚀 Tecnologías Utilizadas

*   **FastAPI:** Framework moderno y asíncrono para construir la API web.
*   **Hugging Face (Transformers):** Uso de modelos de procesamiento de lenguaje natural (NLP) preentrenados.
*   **PyTorch:** Motor de *Machine Learning* subyacente.
*   **Uvicorn:** Servidor ASGI para producción.
